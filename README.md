# 📚 ebook-promotions

一個**給懶人用的電子書平台活動彙整頁**。

👉 線上頁面：  
https://sapling1107.github.io/ebook-promotions/

---

## 這是什麼？

這個頁面存在的唯一目的就是：

> **我不想每天打開 BookWalker、HyRead、Pubu，一個一個看活動。**

所以提出了一堆任性又零碎的需求，  
然後交給 Ashel（ChatGPT）想辦法把它們湊成一個真的能跑的東西。

結果就變成了你現在看到的這個頁面。

它不是比價工具，也不是推薦系統，  
只是單純的 **「活動集中清單」**。

---

## 這個頁面會做什麼？

- 彙整各平台目前可見的活動標題  
- 每日自動更新  
- 標示「🆕」代表與前一天相比新出現的活動  
- 點擊後會回到各平台的官方活動頁  

---

## 這個頁面不會做什麼？

- ❌ 不幫你算折扣哪個最划算  
- ❌ 不推薦你該買哪本書  
- ❌ 不保證活動內容或優惠條件一定準確  

如果你需要精確折扣或購買建議，  
請直接以各平台官方資訊為準。

---

## 目前收錄的平台

- BookWalker  
- HyRead  
- Pubu  

---

## 那 Readmoo 呢？

我其實也很喜歡 Readmoo。

只是很遺憾地，我真的不會寫程式，  
而 Readmoo 的活動頁呈現方式，對現在這個專案來說難度有點太高。

所以目前的狀態是：

> **我不是不想做，是我真的不會。**

如果哪天我學會了，或 Ashel 突然變得更聰明，  
再來考慮補上。

其他平台則因使用頻率與整理方式差異，暫未納入本頁彙整範圍。

---

## 技術說明（非常簡略）

- 由 Ashel 協助撰寫與調整抓取邏輯  
- 使用 GitHub Actions 定期執行  
- 產生靜態 JSON 與 HTML  
- 透過 GitHub Pages 發佈  

本專案沒有後端、沒有資料庫，也沒有帳號系統。

---

## 注意事項（請務必看）

> 本頁僅彙整各平台目前可見的活動標題，用於快速瀏覽與比對是否有新增活動。  
> 活動內容、折扣條件與實際有效期限，請以各平台官方頁面公告為準。
